# 剪刀石頭布獻上小遊戲 DEMO
本專案用於測試小型專案開發。

## 專案目的
完成剪刀石頭布遊戲開發，支援多人遊玩。
* Login Page 登入介面 ( RWD: sm、lg )
* Pick Page 選擇遊戲人員介面 ( RWD: sm、lg ))
* Main Page 主遊戲介面 ( RWD: sm、lg )
* LocalStorage control
* cors 跨域處理

## 使用技術 / Framwork
* HTML
* Javascipt
* Tailwinds
* ES6
* Axios
* react-icon
